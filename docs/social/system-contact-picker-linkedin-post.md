# LinkedIn Post: system_contact_picker

Visual: `public/social/system-contact-picker-linkedin.png`

I published `system_contact_picker: ^0.0.2` on pub.dev.

It is a Flutter package that opens the native system contact picker instead of forcing apps to build and maintain a custom contact-selection screen.

Current platform behavior:
- Android 17 system Contact Picker on API 37+
- Legacy Android contact picker fallback for older devices
- iOS ContactsUI support
